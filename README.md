# Lobami Technologies

Sitio corporativo para presentar Lobami Technologies y enlazar los productos activos:

- Saldy
- Ruta Abierta
- Nudo

## Desarrollo local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

El repositorio incluye un workflow de GitHub Actions para publicar a Cloudflare Pages en cada push a `main`.
